# mcp.quran.ai

This repository will eventually host the public open-source release of the [quran.ai MCP server](https://mcp.quran.ai). For now, it contains publicly available skills.

## Skills

- **[quran.ai MCP Usage](skills/quran.ai-MCP-usage/SKILL.md)** — Operational skill guide for using the quran.ai MCP server's search, fetch, and linguistic tools.
- **[quran.ai MCP Grounding Rules](skills/quran.ai-MCP-grounding-rules/SKILL.md)** — Rules ensuring all Quran text, translations, and tafsir are sourced from canonical data rather than AI memory.
